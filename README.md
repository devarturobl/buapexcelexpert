# 1. Introducción y Contexto del Repositorio

Este repositorio constituye un ecosistema técnico de alto nivel diseñado para dominar los dominios del examen **MO-211: Microsoft Excel Expert**. A través de una metodología de simulación empresarial inmersiva, el usuario gestiona flujos de trabajo complejos que replican escenarios reales de consultoría y análisis de datos.

La relevancia estratégica del nivel *Expert* radica en la transición de:

- Procesamiento reactivo de datos  
→ a  
- Arquitectura proactiva de inteligencia de negocios  

Un experto diseña sistemas automatizados y modelos predictivos que optimizan la toma de decisiones.

### Componentes del repositorio

- 📂 **Archivos de Datos Maestros**
  - `Project1_datafile.xlsx`

Actuar como consultor técnico para:

## 3. Proyecto 1: Gestión de Datos y Funciones Lógicas

Enfocado en transformar datos en información confiable.

### Competencias clave

| Competencia | Aplicación |
|------------|----------|
| Validación de datos | Restricción de valores (0–3), alertas y detección de errores |
| Formato | IDs personalizados (`ID-0000`) y formato de fechas (`mmmm`) |
| Lógica dinámica | `SWITCH`, `IF`, `HLOOKUP` |
| Histogramas | Análisis de frecuencia de ventas |
| Tablas dinámicas | Agrupación por mes y trimestre |
| Proyecciones | Uso de `RANDARRAY` |

## 4. Proyecto 2: Análisis Avanzado y Seguridad

Se centra en visualización, integración y protección de datos.

### Hitos técnicos

- 🔄 **Power Query**

- 📊 **Visualización avanzada**
  - Funnel (Embudo)

  - Box & Whisker (Cajas y bigotes)

  - Búsquedas robustas con manejo de errores

  - Protección con contraseña: `password`
  - Restricción de rangos (ej. `QuantitySold`)
