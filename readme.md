📖 About this Study Guide
Bienvenido a la guía de preparación para el examen MO-211: Excel Expert (Microsoft 365 Apps). Esta guía está diseñada para que educadores ayuden a sus alumnos a dominar los dominios avanzados de Excel.

Study Guide Features
Escenario Único: Utiliza el caso de negocio de Munson’s Pickles and Preserves Farm.

Proyectos Prácticos: Tres proyectos de 50 minutos cada uno (simulando el examen real).

Learning Directory: Tabla de objetivos con recursos online y guía paso a paso.

🏗️ Structure & Projects
Proyectos de Examen
Cada proyecto incluye hasta 40 tareas a completar en 50 minutos.

Proyecto 1: Gestión de datos y funciones lógicas.

Proyecto 2: Análisis avanzado y seguridad.

Proyecto 3: Automatización y modelado financiero.

Data Files & Solutions
Los materiales incluyen archivos de inicio y soluciones en tres formatos:

Project_StepbyStepSolution.docx: Instrucciones detalladas.

Project_VideoStepbyStepSolution.pptx: Demostración visual (sin audio).

Project_SolutionFile.xlsx: El archivo final completado con éxito.

💡 Instructional Methods and Strategies
Simulación Real: Se recomienda cronometrar a los alumnos para simular el entorno del examen.

Uso de Checkboxes: Fomentar el uso de casillas de verificación para rastrear el progreso.

Diferenciación: Los alumnos pueden comparar sus archivos con las soluciones o ver los videos si se quedan atascados.

Learning Directory: Herramienta clave para la remediación de habilidades específicas que no se dominan.

Tip para el aula: Si se dispone de dos monitores, se recomienda seguir la guía en uno y realizar las actividades en el otro.

💻 Software Requirements
Para garantizar la compatibilidad, los estudiantes deben contar con:

Microsoft 365 Apps (Versión de escritorio).

El examen MO-211 se basa específicamente en las funciones de la suscripción Microsoft 365.
